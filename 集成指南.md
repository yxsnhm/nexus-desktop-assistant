# Nexus 桌面助手 - 模块集成指南

## 概述

桌面助手可作为模块集成到智联枢纽主程序或其他 Python 项目中，实现远程调用、任务派发等功能。

## 快速集成

### 1. 导入并启动

```python
from nexus_tool import NexusTool

# 非阻塞启动，可选回调处理机器人结果
def my_callback(cmd, result):
    print(f"指令: {cmd}, 结果: {result}")

app = NexusTool.start_as_module(callback=my_callback)
2. 主要接口
日志输出：app.log("消息") — 在主窗口日志区显示消息

获取系统状态：app._get_system_state() — 返回窗口状态、调度池大小等

执行远程命令：app._execute_remote_command(json_str) — 通过 JSON 下发指令

发送消息到窗口：app._do_send_and_capture("千问", "消息内容", False)

启动调度：app._start_schedule()

停止任务：app._stop_task()

3. 示例：远程派发任务
python
import json

# 启动调度
app._execute_remote_command(json.dumps({
    "action": "schedule_start",
    "windows": ["千问", "DeepSeek"],
    "topic": "讨论人工智能发展"
}))

# 查看状态
state = app._get_system_state()
print(state)
配置文件
桌面助手默认读取 D:\智联枢纽\nexus_config.json，可通过修改 NexusTool.CONFIG_FILE 类变量自定义路径。

注意事项
桌面助手 GUI 必须在主线程中运行，start_as_module 内部会创建新线程

所有窗口操作基于坐标和关键词，确保配置文件正确

机器人执行器已包含安全沙箱，exit() 等危险操作被拦截
