# Nexus 桌面助手 - 使用指南

## 一、下载与安装

### 方式一：直接下载（推荐）

访问项目仓库的 Releases 页面，下载以下两个文件：
- `Nexus桌面助手.exe` — 主程序
- `NexusRobot.exe` — 内置机器人窗口

将两个文件放在**同一个文件夹**中，双击 `Nexus桌面助手.exe` 即可启动。

### 方式二：从源码运行

1. 克隆或下载项目源码
2. 安装依赖：`pip install -r requirements.txt`
3. 运行：`python nexus_tool.py`

---

## 二、首次使用配置

### 1. 准备配置文件

将 `nexus_config.json.example` 复制一份，重命名为 `nexus_config.json`。

### 2. 打开你的 AI 窗口

在浏览器中打开你要使用的 AI 网页（如千问、DeepSeek、智谱等），确保窗口标题栏包含可识别的关键词。

### 3. 校准窗口位置

启动桌面助手后，切换到“🎯 定位校准”标签页：

1. 点击 **🎯 校准**：在 5 秒内将鼠标放到 AI 窗口的输入框上
2. 点击 **📖 校准读取区**：将鼠标放到 AI 窗口的聊天内容区域
3. 点击 **📋 校准复制键**：将鼠标放到 AI 窗口的复制按钮上
4. 点击 **🔍 测试捕获**：验证校准是否成功

每个窗口都需要单独校准。

### 4. 添加新窗口

点击 **➕ 添加窗口**，输入：
- 窗口名称：任意名称（如“千问”）
- 监控方式：截图监控（适合千问/智谱等有复制键的）或全选监控（适合 DeepSeek 等）
- 窗口关键词：窗口标题中包含的文字，用于定位

## 三、功能使用

### 💬 对话模式

1. 切换到“💬 对话”标签页
2. 在顶部选择目标窗口
3. 输入内容，按回车或点发送
4. AI 回复会自动抓取并显示

### 🔄 智能调度

1. 切换到“🔄 智能调度”标签页
2. 勾选参与讨论的窗口
3. 输入初始话题
4. 点击“启动智能调度”

### @指定窗口

在对话页输入 `@窗口名 消息内容`，可将消息定向发送到指定窗口。

例如：`@千问 你好，请介绍你自己`

### 🤖 机器人指令

桌面助手内置 Python 执行引擎，输入 `@ 代码` 可执行指令。

例如：
- `@ print("hello")`
- `@ import os; os.system("start notepad")`

### 📱 移动控制台

手机浏览器访问 `http://电脑IP:5000`，可远程：
- 查看系统状态
- 发送指令
- 启动/停止调度

## 四、常见问题

**Q：窗口切换不了？**
A：检查配置文件中的 `keyword` 是否与窗口标题匹配，重新校准坐标。

**Q：抓取不到回复？**
A：检查读取区坐标是否对准聊天内容区，或尝试切换监控方式。

**Q：移动端连接不上？**
A：确保手机和电脑在同一 WiFi，检查防火墙是否开放 5000 端口。

**Q：配置丢失了怎么办？**
A：将 `nexus_config.json.example` 复制为 `nexus_config.json`，重新校准即可。

## 五、配置文件参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `screenshot_interval` | 截图监控检测间隔(秒) | 3 |
| `stable_wait` | 稳定后等待时间(秒) | 3 |
| `idle_timeout` | 超时跳过轮数 | 10 |
| `scroll_count` | 每次滚轮滚动次数 | 10 |
| `scroll_amount` | 每次滚动像素量 | 500 |
| `click_max` | 复制键最多点击次数 | 2 |
| `mobile_port` | 移动端服务端口 | 5000 |
